الويب: HTML • CSS • JavaScript

أساس بناء مواقع الويب الحديثة. ابدأ بـ HTML للهيكل، CSS للتصميم، ثم JS للتفاعل والمنطق.

HTML

مقدمة وأساسيات HTML

الوسوم، البنية الدلالية، العناوين، الفقرات، الروابط، الصور، القوائم.

<!DOCTYPE html>
<html lang="ar" dir="rtl">
  <head> <meta charset="UTF-8"> <title>صفحة</title> </head>
  <body>
    <h1>مرحبًا</h1>
    <a href="/about">من نحن</a>
  </body>
</html>
  • وسوم مهمة: <header>, <nav>, <main>, <section>, <article>, <footer>
CSS

تصميم و Responsive Layout

Flexbox, Grid, ألوان متغيرة، أنيميشن وانتقالات.

/* شبكة بسيطة */
.container { display: grid; grid-template-columns: repeat(12, 1fr); gap: 1rem; }
.card { grid-column: span 4; }
@media (max-width: 768px){ .card { grid-column: span 12; } }

نصائح: استخدم متغيرات CSS، وأعد استخدام المكوّنات، وانتبه للتباين.

JavaScript

التفاعل والمنطق

DOM، الأحداث، الوعود، async/await، وحدات ES.

// مثال جلب بيانات
async function loadUsers(){
  const res = await fetch('https://jsonplaceholder.typicode.com/users');
  const data = await res.json();
  console.log(data);
}
loadUsers();

جرّب التعامل مع LocalStorage و APIs و Modular JS.

لغات البرمجة الأساسية: C++ • Python • Java • C#

مقارنة سريعة وأمثلة عملية لأكثر اللغات طلبًا في السوق.

C++

لماذا C++؟

أداء عالٍ، التحكم بالذاكرة، الألعاب والأنظمة.

#include <iostream>
using namespace std;
int main(){
  cout << "Hello C++";
  return 0;
}

المواضيع: المؤشرات، OOP، القوالب، STL، إدارة الذاكرة.

Python

لماذا Python؟

بسيطة وسريعة للتطوير، علم البيانات والذكاء الاصطناعي.

def greet(name: str) -> str:
    return f"Hello {name}"
print(greet("Youssef"))

المواضيع: مكتبات مثل requests, pandas, numpy, أساسيات OOP.

Java

لماذا Java؟

قوية للمشاريع الكبيرة، Android، Spring.

class Main { public static void main(String[] a){ System.out.println("Hello Java"); } }

المواضيع: JVM، المجموعات، الخيوط (Threads)، Spring Boot.

C#

لماذا C#؟

تطبيقات سطح المكتب، الويب (ASP.NET)، الألعاب (Unity).

using System; class App{ static void Main(){ Console.WriteLine("Hello C#"); } }

المواضيع: LINQ، async/await، Entity Framework.

الخلفية: PHP • Node.js

خادوميات، REST APIs، قواعد البيانات، المصادقة.

PHP

REST API بسيط

<?php
header('Content-Type: application/json');
$users = [["id"=>1,"name"=>"Ali"],["id"=>2,"name"=>"Sara"]];
echo json_encode($users);

إطار عمل مقترح: Laravel. قاعدة بيانات: MySQL.

Node.js

Express + JWT

import express from 'express';
const app = express();
app.get('/api/hello', (req,res) => res.json({msg:'Hi'}));
app.listen(3000);

إطار عمل: Express/NestJS. قاعدة بيانات: MongoDB/PostgreSQL.

React وواجهات حديثة

مكوّنات قابلة لإعادة الاستخدام، حالة (State)، ربط APIs.

React

مكوّن بسيط

function Hello({name}){ return <h1>Hello {name}</h1> }

جرّب Next.js للتوليد المسبق والـ SEO، وTailwind للتنسيق السريع.

Arduino والإلكترونيات

ابدأ بالـ IDE، افهم الـ Pins، جرّب الحساسات والمحركات.

Arduino

تشغيل LED

int led = 13;
void setup(){ pinMode(led, OUTPUT); }
void loop(){ digitalWrite(led, HIGH); delay(500); digitalWrite(led, LOW); delay(500); }

استخدم المقاومات، وتعلّم قراءة الداتا شيت للحساسات.

Sensors

قراءة حساس (مثال عام)

// pseudo: يختلف حسب الحساس والمكتبة
#include <SomeSensor.h>
SomeSensor s;
void setup(){ Serial.begin(9600); s.begin(); }
void loop(){ Serial.println(s.read()); delay(1000); }

جرّب DHT11/22 لدرجة الحرارة، وHC-SR04 للموجات فوق الصوتية.
